Para ver la memoria completa y resultados del proyecto "Ciudad vs. Pueblo", hay que abrir los siguientes archivos:

1. memoria_proyecto_II.html contiene el grueso de la memoria. Es el HTML principal
2. app/memoria_app.html contiene el contenido al que no llega la memoria principal porque está estructurada para generar una aplicación en RStudio.
3. app/script_app.Rmd es el R Markdown que crea la aplicación para que se pueda usar libremente.
	- Instalar las librerías sf, leaflet, dplyr, stringr, redr, htmltools, shiny, bslib y tidr.
	- Ejecutar el archivo de arriba a abajo y esperar a que se abra una ventana emergente con la app.
